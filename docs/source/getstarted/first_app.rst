First application
=================

1. Import ``Charmy``

.. code-block:: python

   import charmy as cm

2. Create a ``Window``

.. code-block:: python

   window = cm.Window()

3. Run the application
There is auto created ``App`` object. You can run it by ``cm.mainloop()``.

.. code-block:: python

   cm.mainloop()

In all, your first application should look like this:

.. code-block:: python

   import charmy as cm

   window = cm.Window()
   cm.mainloop()

Then, you can get a window.
